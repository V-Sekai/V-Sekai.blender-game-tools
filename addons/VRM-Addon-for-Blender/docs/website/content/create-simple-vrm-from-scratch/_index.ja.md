Blenderを起動したときにはじめから表示されている立方体をそのままVRMモデルにします。

![](simple.gif)

まずはBlenderを起動し、立方体が表示されていることを確認します。

![](1.ja.png)

次にメニューの `ファイル` → `エクスポート` → `VRM (.vrm)`を選択します。

![](2.ja.png)

ファイル保存用のウィンドウが出るので、ファイル名と保存先を入力し `Export VRM` を押します。

![](3.ja.png)

成功するとVRMファイルが指定された場所に保存されます。

![](simple.gif)

こちらのページで動作確認ができます。
